## 🚀 Unified Intelligence Overview

**AetherFlow** is a premium, data-centric Supply Chain Management system engineered for industrial complexity. It features a **Universal Analytics Engine** and a **Neural Simulation Layer** that allows executives to stress-test global fulfillment networks in real-time.

### 🛡️ Enterprise Governance & RBAC
- **Admin Clearance**: Full access to the Neural SQL Console and Universal Data Ingestion Hub.
- **Analyst Access**: Restricted view for standard employees, featuring auto-refreshing intelligence grids but locked direct-write protocols.

### 📊 Intelligence Pillars
- **Universal Ingestion**: Seamless support for binary **Excel (.xlsx)** and **CSV** payloads with automated schema inference.
- **Digital Twin Exporting**: One-click "Export Report" engine allowing stakeholders to download analytical snapshots in JSON format for external auditing.
- **Neural Grid Resilience**: Optimized for Vercel with a "Pre-Baked Database" template, ensuring instant 1-second startup and 100% data availability on cold starts.
- **Neural Simulation**: A "What-If" Stress Test engine that simulates demand shocks (300%+ spikes) to calculate brittle supply nodes.

## 🛠️ Technology Stack

- **Core Engine**: Next.js 16 (Turbopack), React 19, Lucide React.
- **Intelligence Layer**: SQLite3 (better-sqlite3) with **17 Normalized Tables** and **4 Active Triggers**.
- **Governance**: "Obsidian & Lime" Design System with high-contrast industrial ergonomics.
- **Deployment**: Vercel ready with ephemeral filesystem bypass for database integrity.

## 📐 Database Architecture (17 Tables)

The system operates on an interconnected grid of **3,200+ live records**:

| Layer | Functional Tables |
| :--- | :--- |
| **Foundation** | `Categories`, `Suppliers`, `Manufacturers`, `Distributors`, `Warehouses` |
| **Product Stack** | `Products`, `ProductSuppliers` (M:N Mapping) |
| **Physical Flow** | `Orders`, `Logistics` (Auto-Triggered), `InventoryTransactions` |
| **Intelligence** | `InventoryLevels`, `QualityChecks`, `MaintenanceLogs` |
| **Governance** | `Users` (RBAC), `AuditLogs`, `SupplierPerformanceLogs`, `InventorySnapshots` |
